src/kmeans.js

K-means algorithm

kmeans(data: Array<Array<Number>>, centers: Array<Array<Number>>, options: [Object]): Object
Parameters
data (Array<Array<Number>>) the (x,y) points to cluster
centers (Array<Array<Number>>) the K centers in format (x,y)
options ([Object]) properties
Name Description
options.maxIterations [Number] (default 100) maximum of iterations allowed
options.tolerance [Number] (default 1e-6) the error tolerance
options.withIterations [boolean] (default false) store clusters and centroids for each iteration
Returns
Object: the cluster identifier for each data dot and centroids
