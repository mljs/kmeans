src/kmeans.js

K-means algorithm

kmeans(data: Array<Array<Number>>, K: Number, options: [Object]): Object
Parameters
data (Array<Array<Number>>) Points in the format to cluster [x,y,z,...]
K (Number) Number of clusters
options ([Object]) Option object
Name Description
options.maxIterations [Number] (default 100) Maximum of iterations allowed
options.tolerance [Number] (default 1e-6) Error tolerance
options.withIterations [Boolean] (default false) Store clusters and centroids for each iteration
options.initialization [(String | Array<Array<Number>>)] (default 'moreDistant') K centers in format [x,y,z,...] or a method for initialize the data:
  • 'random' will choose K random different values.
  • 'moreDistant' will choose the more distant points to a first random pick
Returns
Object: Cluster identifier for each data dot and centroids with the following fields:
  • 'clusters': Array of indexes for the clusters.
  • 'centroids': Array with the resulting centroids.
  • 'iterations': Array with the state of 'clusters' and 'centroids' for each iteration.
